# Análisis de movilidad urbana y productividad económica en LATAM

## Objetivo
Analizar la relación entre congestión vehicular y productividad económica en ciudades de América Latina utilizando datasets reales.

## Herramientas utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Proceso realizado
- Limpieza y validación de datos
- Unificación de datasets
- Análisis Exploratorio de Datos (EDA)
- Visualización de tendencias y patrones

## Insights obtenidos
- Se identificaron ciudades con altos niveles de congestión y menor productividad económica.
- El análisis permitió detectar patrones útiles para la toma de decisiones e inversión en infraestructura urbana.

## Dataset
- TomTom Traffic Index
- OECD Cities
